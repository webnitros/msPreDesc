Описание

Компонент предназначен для массового управление ресурсами/товарами для MODX revolution, поддерживаемые версии начиная с 2.3 и до 3-й версии modx включительно. Через компонент можно управлять тысячами ресурсов одновремено.

Легко выдерживает большое количество товаров за счет пакетной обработки.

Требования

  • Modx Revolution 2x/3x
  • PHP 5.4+

Возможности

Основные возможности компонента это фильтрация и массовое управление товарами и ресурсами

Управление

Управление большим объемом товаров
Можно установить значение сразу для всех отмеченных товаров или для всех товаров указанных на странице
Дерево ресурсов
Быстрая навигация по всем категориям и фильтрация товаров
Увеличение изображений
Удобное переключение между миниатюрами размеров 40x40 к размеру 120x90
Модификации msOptionsPrice2
Редактируйте цены и создавайте новые модификации от компонент msOptionsPrice2
Цены
Увеличивайте цены на проценты или прибавляйте указанную сумму
ТВ-параметры (Дополнительные поля)
Добавить/Изменить/Удалить значение у выбранного тв-параметр, с указанием для каких шаблонов потребуется изменения
Опции miniShop2
Редактирование опций miniShop2

Фильтры

Показывать товары из дополнительных категорий
При выборе категории с лева, вы смоежет найти товары которые попали в эту категориию через вкладку "Категории"
Показывать купленные товары
Получить список товаров которые когда то покупали можно с помощью фильтра "Показывать купленные товары". Статусы заказов указываются в настройках компонента
Модификации
Найти все товары с модификациями msOptionsPrice2
Фильтрация по любому из полей в базе данных
С возможностью выбрать оператор для поиска
